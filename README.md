# Wordpress

This is an example Wordpress project built with Bedrock and Composer for dependency management. The bedrock wordpress is a solid foundation, so the changes are minimal - you can make your blog in minutes, not days. 

Checkout [Wordpress Bedrock Readme](./README-bedrock.md) for getting started!

## Deploying with DollarDeploy

You don't need much to configure it to deploy with DollarDeploy, follow this section to get things running quickly:

### Configure host:

* Add service MariaDB and prepare the host. This will set MARIADB_URL service variable.

### Add new app:

* Details => Application type: php
* Deploy => Custom pre-start command: sudo apt install -y php-fpm php-pgsql php-mysql
* Deploy => Web path: web

### Deploying on command line

```bash
npm install -g @dollardeploy/cli@latest
DOLLARDEPLOY_API_KEY=... ddc --url https://github.com/dollardeploy/example-
wordpress --provider datacrunch --services mariadb \
  --app:preStartCmd "sudo apt install -y php-fpm php-pgsql php-mysql" \
  --app:type php --app:webPath web --app:name wordpress
```

### App Environment

* DATABASE_URL=${MARIADB_URL}
* WP_ENV=development
* WP_HOME=${APP_URL}
* WP_SITEURL=${WP_HOME}/wp

Add additional security settings as generated by [Wordpress salts generator](https://roots.io/salts.html)
